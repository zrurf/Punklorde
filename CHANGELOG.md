# 更新日志 | Changelog

## [`75ad3de`](https://github.com/zrurf/Punklorde/commit/75ad3dee9a7cc66811b52a5d84203f7db1525b95) - 2026-3-29

### 🚨 重大破坏性改动 | Breaking Changes

- **项目完全重构** ([`2434c60`](https://github.com/zrurf/Punklorde/commit/2434c607959b383ba2258dffee06fdfafba57d93) 之后)
    - 存储模型完全重构，**与之前版本完全不兼容**。
    - 分享码机制调整，旧版分享码无法使用。
    - 登录使用的 OpenID 已改变。
    - **操作建议**：必须**完全卸载旧版本**后再安装新版本，无法通过直接覆盖安装升级。旧数据无法迁移。
- **弃用同天智慧体育** (自 2026.03.02 起)
    - 由于外部平台（同天智慧体育）被弃用，`2434c60` 及其之前所有构建版本已**完全失效**。

### ✨ 新增 | Added

- **核心架构重构**：引入了Rust和Go，部分模块将逐渐迁移到Rust或Go。
- **基础框架**：搭建了多学校/多平台适配的基底框架。

### 计划中
- 课表模块基础 UI。
- 学习通签到支持
