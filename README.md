## 构建
需要**Flutter** `3.38` 及以上版本，**Android SDK** `36` 及以上版本，**Android NDK** `r27` 及以上版本。构建 **Android** 时， `compileSdkVersion` 至少为`35`。

构建前需要在 `.env` （参考 `.env.template` ）和 `android/local.properties`（参考`local.properties.template`） 中填写您的Key。您应当自行申请相关API Key。

构建产物目前支持 **Android 8** (`API 26`) 及以上版本。iOS缺乏维护和测试，欠缺部分功能，可构建，但可能不能稳定运行。

### 签名
Android请自行使用Keytool生成证书来签名，需填写`android/key.properties`（参考`key.properties.template`）。证书与百度地图Api Key相关，请确保正确配置。

iOS请使用个人开发者证书签名。
