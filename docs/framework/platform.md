# 支持的平台
本项目优先考虑移动设备，但是部分功能有桌面平台需求，因此会支持部分桌面平台。

**Android**（arm64-v8a, API 26）和**Windows**（Windows 11，25H2）是主要开发平台，也是新功能实验平台，因此新功能会优先支持**Android**，其余平台会尝试迁移。

请注意，该支持矩阵不同于Flutter的支持矩阵，因为该项目使用了Native代码。

以下是支持矩阵：

> ✅: 完整支持 <br/>
> 🚧: 支持，但未经测试 <br/>
> ⚠: 支持，但存在问题，可能无法编译或运行 <br/>
> ❌: 暂不支持 <br/>

> **Tier 1**: 优先支持，持续维护 <br/>
> **Tier 2**: 次要支持，维护不积极 <br/>
> **Tier 3**: 有相关计划，但暂不维护 <br/>

| 平台 | 架构 | 支持程度 | 支持等级 | 备注 |
| :---:| :---: | :---: | :---: | :---: |
| **Android** | **arm64-v8a** | ✅ | `Tier 1` | 优先支持，也是新功能的实验平台 |
| **Android** | **armeabi-v7a** | 🚧 | `Tier 2` | |
| **Android** | **x86_64** | 🚧 | `Tier 2` | |
| **iOS** | **arm64** | 🚧 | `Tier 1` | 部分功能可能不支持 |
| **Windows** | **x86_64** | ⚠ | `Tier 3` | |
| **Windows** | **arm64** | ❌ | `-` | 暂不考虑 |
| **macOS** | **arm64** | ⚠ | `Tier 3` | |
| **macOS** | **x86_64** | ❌ | `Tier 3` | 暂不考虑 |
